# Vue组件库

element-ui (opens new window): Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

vant (opens new window): Vant 是有赞前端团队开源的移动端组件库，于 2017 年开源，已持续维护 4 年时间。Vant 对内承载了有赞所有核心业务，对外服务十多万开发者，是业界主流的移动端组件库之一。

DataV (opens new window): Vue数据可视化组件库（类似阿里DataV，大屏数据展示），提供SVG的边框及装饰、图表、水位图、飞线图等组件，简单易用，长期更新(React版已发布)

Ant Design of Vue (opens new window): Ant Design 的 Vue 实现

Vue.Draggable (opens new window): 基于Sortable.js (opens new window)的Vue拖放组件

svgicon (opens new window): SVG 图标组件和工具集 官网 (opens new window)

vue-luck-draw (opens new window): 一个基于 vue2 / vue3 的【大转盘 / 九宫格】抽奖插件，基于lucky-canvas (opens new window)

vue-baberrage (opens new window): 基于Vue.js弹幕插件.

# Vue指令

v-animated (opens new window): Vue.js2.X动画指令插件,此插件基于animate.css4.x和wow.js实现

v-track (opens new window): 一个基于Vue指令实现的埋点解耦插件

# js库

wow (opens new window): 向下滚动页面时显示CSS动画。默认情况下，您可以使用它来触发Animate.css (opens new window)动画。但是您可以轻松地将设置更改为您喜欢的动画库。

js-export-excel (opens new window): 纯 js 导出 excel

better-scroll (opens new window): BetterScroll是一个插件，旨在解决移动端的滚动情况（已支持PC）。核心灵感来自iscroll的实现，因此BetterScroll的API总体上与iscroll兼容。此外，BetterScroll还扩展了某些功能，并基于iscroll优化了性能。

lodash (opens new window): Lodash 是一个一致性、模块化、高性能的 JavaScript 实用工具库。官网 (opens new window)

mitt (opens new window): 事件分发，监听触发工具库

move.js (opens new window): CSS3 JavaScript动画框架。

# css样式库

tailwindcss (opens new window): 无需编写css，即可高度自定义编写出优美界面的css库

tailwindgrids (opens new window): 生成基于tailwindcss的grids布局网站

Animate.css (opens new window): css动画样式库

NES.css (opens new window): 是一个NES样式（类似小时候小霸王游戏里的字体）的CSS框架。官网 (opens new window)

magicpattern (opens new window): css背景图案

# 微信小程序

vant weapp (opens new window): 轻量、可靠的小程序 UI 组件库

ColorUI (opens new window): 鲜亮的高饱和色彩，专注视觉的小程序组件库

# 富文本编辑器

quill (opens new window): web富文本编辑器

wangeditor (opens new window): 轻量级web富文本框

# vue生态

nuxt.js (opens new window): Nuxt.js 是一个基于 Vue.js 的通用应用框架。SSR，SEO优化

electron-vue (opens new window): electron + vue构建桌面应用

ice (opens new window): 飞冰，简单而友好的前端研发体系

Vant Use (opens new window): Vant Use 是 Vant 团队提供的 Vue Composition API 库。

vue-awesome-swiper (opens new window): Swipe 轮播（你想要的轮播基本都有）

vuescroll (opens new window): 一个基于vue.js的可自定义的滚动条插件，适用于PC，手机，触摸屏，笔记本电脑。

v-contextmenu (opens new window): 基于Vue 2.0的ContextMenu菜单组件

vue-seamless-scroll (opens new window): vue无缝滚动组件

# React

react-loadable (opens new window): React组件异步加载

react-hooks (opens new window) React Hooks 学习资料

# 实用工具

YApi (opens new window): 旨在为开发、产品、测试人员提供更优雅的接口管理服务。可以帮助开发者轻松创建、发布、维护 API

vite (opens new window): 下一代前端开发工具。它很快。

vuepress (opens new window): Vue提供的静态站点生成器。

vitepress (opens new window): Vite＆Vue提供的静态站点生成器。与vuepress的区别是，vitepress更快。

# 其他

cool-admin-midway (opens new window): cool-admin(midway版)一个很酷的后台权限管理框架，模块化、插件化、CRUD极速开发，永久开源免费，基于midway.js 2.0、typeorm、mysql、jwt、element-ui等构建

cool-admin-vue (opens new window): cool-admin一个很酷的后台权限管理框架，模块化、插件化、CRUD极速开发，永久开源免费，基于midway.js 2.0、typeorm、mysql、jwt、element-ui、vuex、vue-router、vue等构建

recorder (opens new window): html5 js 浏览器 web端录音

single-spa (opens new window): Single-spa 是一个将多个单页面应用聚合为一个整体应用的 JavaScript 微前端框架。

微前端架构 (opens new window): 每日优鲜供应链前端团队微前端改造 扩展阅读 (opens new window)

前端微服务示例项目仓库 (opens new window)

qiankun (opens new window): qiankun 是一个基于 single-spa (opens new window) 的 微前端 (opens new window) 实现库，旨在帮助大家能更简单、无痛的构建一个生产可用微前端架构系统。

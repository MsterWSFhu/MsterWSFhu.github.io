CSS3@font-face不生效问题

sifu   in 南昌
2020-12-31 Thursday
  • CSS3
  • font-face
  • 字体

# 1、加载字体

注意：字体文件名称 zpix 和 font-family中的值 'zpix' 需要一致

@font-face {
    font-family: 'zpix'; 
    src: url('/blog/assets/font/zpix.ttf');
}

/*以下写法是无效的*/
@font-face {
    font-family: 'font-zpix'; 
    src: url('/blog/assets/font/zpix.ttf');
}
